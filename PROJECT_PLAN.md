# 建設会社AIプロジェクト 計画

このプロジェクトでは、建設会社の業務を支えるAIを4つの柱で整理します。

## 1. AI秘書

社長、役員、管理者、事務担当者の毎日の仕事を支援するAIです。

### やりたいこと

- メールや文章の下書き作成
- スケジュール整理
- 会議メモの作成
- ToDoリスト作成
- 社内連絡文の作成
- 顧客対応文の作成
- 書類作成の補助

### 最初の試作品

- 文章作成AI
- 会議メモ整理AI
- 今日やること整理AI

## 2. 積算AI

見積もり、数量拾い、単価整理、原価確認を支援するAIです。

### やりたいこと

- 見積書の下書き作成
- 工事項目の整理
- 数量拾いの補助
- 単価表の整理
- 過去見積との比較
- 原価と粗利の確認
- 抜け漏れチェック
- PDF図面からの数量拾い出し
- 拾い出した数量の工種別整理
- 拾い出し根拠の保存

### 最初の試作品

- PDF図面ビューア
- 縮尺設定
- 距離計測
- 面積計測
- 数量リスト作成
- 見積明細への反映

### 既存資産

`F:\Dropbox\OPENAI` に以前作成した見積りアプリがあります。

- `mitsumori_app/index.html`
- `mitsumori_app/app.js`
- `mitsumori_app/import_files.py`
- `mitsumori_data.json`

確認済みの土台:

- 見積作成
- 工種別見積
- PDF / Excel / CSV 取り込み
- PDF出力
- Excel出力
- コンクリート、土工、型枠、左官、金属などのテンプレート

今後はこの既存アプリをベースに、PDF図面の拾い出し機能を追加していきます。

## 3. 現場AI

現場監督、職人、施工管理を支援するAIです。

### やりたいこと

- 日報作成
- 作業報告の整理
- 工事写真の整理
- 安全書類の作成補助
- 工程表の確認
- 現場の注意事項整理
- 協力会社への連絡文作成

### 最初の試作品

- 日報作成AI
- 工事写真メモ整理AI
- 現場連絡文作成AI

## 4. 経営AI

会社全体の数字、案件、利益、人員、方針を見える化するAIです。

### やりたいこと

- 売上、利益、原価の整理
- 案件ごとの状況確認
- 資金繰りメモの整理
- 経営会議資料の下書き作成
- 月次レポート作成
- 問題案件の早期発見
- 採用や人員配置の検討補助

### 最初の試作品

- 月次レポートAI
- 案件状況まとめAI
- 利益チェックAI

## 優先順位

まずは積算AIから進めます。

1. 積算AI
2. 現場AI
3. AI秘書
4. 経営AI

## 最初のゴール

最初は大きなシステムを作るのではなく、既存の見積りアプリに小さく機能を追加します。

- 文字化けを直す
- PDF図面を表示する
- 縮尺を設定する
- 距離と面積を拾い出す
- 数量を工種別に整理する
- 見積明細へ反映する

この小さな機能を積み重ねて、建設会社専用の積算AIに育てていきます。
