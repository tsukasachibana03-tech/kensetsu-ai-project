`F:\Dropbox\OPENAI-2\tmp\pdfs` から引き継いだPDF画像化サンプルです。

PDF図面・見積PDFの取り込み、画像表示、OCR、拾い出し機能の検証に使います。

## 含まれるファイル

- `fukuhara_kogyo-1.png`
- `noborikawa_quote-1.png`
- `noborikawa_quote-2.png`
- `noborikawa_quote-3.png`
- `tanahara_uehara-1.png`

- PDFを画像として表示する検証
- OCR読み取りの検証
- PDF図面積算機能のサンプルデータ
- 見積PDFから明細候補を拾う検証
