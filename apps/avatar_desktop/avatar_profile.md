# Chibaten AI Secretary Avatar Profile

## Adopted Avatar

Name: Field Support Secretary

Role:
- Main candidate for the desktop AI assistant avatar.
- Construction-office and field-support assistant.
- Handles schedule checks, estimates, quantity takeoff support, file reminders, and light conversation.

Visual Direction:
- Anime-inspired 3D desktop mascot style.
- Young, energetic, friendly, and professional.
- Brown ponytail, teal/white work jacket, headset, tablet, utility pants, work sneakers.
- Suitable for construction business support.
- Current visual level: polished style-up version with cleaner hair detail, richer jacket materials, and a more premium desktop AI secretary look.

Voice Direction:
- Slightly younger than the current Chappy voice.
- Clear, bright, calm, and not too loud.
- Natural Japanese with a little soft Okinawan warmth, but not heavy dialect.
- Current prototype voice: Windows Japanese desktop voice with right-click voice menu and startup greeting.

Desktop Behavior:
- Starts in the lower-right area of the screen.
- Always-on-top floating avatar.
- Draggable by mouse.
- Right-click menu for resize and close.
- Natural fixed-position display: stable expression changes without swaying or fade jitter.
- Switchable expressions: stable face-only normal, smile, closed smile, surprised, soft, and laugh.
- Voice lines: greeting, start work, confirmed, break, voice test, and custom typed text.

Next 3D Production Steps:
- Create front, side, and back reference sheets.
- Convert the design into a VRM-compatible 3D model.
- Add blink, mouth movement, expressions, and idle motions.
- Connect speech and ChatGPT responses.
