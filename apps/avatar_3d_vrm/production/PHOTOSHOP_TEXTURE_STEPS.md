# Photoshop Texture Steps

Use Photoshop to move the clothing closer to the adopted reference image.

## Input

Start from VRoid clothing textures exported from the selected model.

Store working files here:

`production/textures/`

## Jacket Direction

Target look:
- teal outer jacket
- white or light gray front panels
- black inner shirt
- zipper line down the center
- pocket lines on both sides
- small sleeve badge
- ID card or small chest tag

## Color Palette

Use these as a starting point:

- teal: `#238C98`
- deep teal shadow: `#17616B`
- light gray panel: `#E6EDF0`
- dark inner shirt: `#171C20`
- yellow badge accent: `#D9B13B`
- blue-gray tablet case: `#2D4656`

## First Texture Pass

1. Duplicate the exported clothing texture.
2. Keep the original as backup.
3. Paint teal sleeves and side panels.
4. Paint light gray or white front panels.
5. Add a dark inner-shirt area around the chest.
6. Add a zipper line.
7. Add pocket seams.
8. Add one small badge on the upper sleeve.
9. Export as PNG.
10. Import back into VRoid or Blender.

## File Names

Recommended working names:

- `jacket_texture_v1.psd`
- `jacket_texture_v1.png`
- `pants_texture_v1.psd`
- `pants_texture_v1.png`
- `shoes_texture_v1.psd`
- `shoes_texture_v1.png`
