# VRM Creation Steps

## Option A: VRoid Studio

Best for a fast first 3D avatar.

1. Install VRoid Studio.
2. Create a new female model.
3. Match the character:
   - Brown ponytail
   - Soft anime face
   - Headset-like accessory if possible
   - Teal/white workwear-inspired outfit
4. Export as `.vrm`.
5. Save as:

   `F:\Dropbox\OPENAI\kensetsu-ai-project\apps\avatar_3d_vrm\models\chibaten_field_secretary.vrm`

## Option B: Blender

Best for a more accurate custom model.

1. Create or import a humanoid base mesh.
2. Model jacket, headset, tablet, shoes, and utility pockets.
3. Rig as humanoid.
4. Add expressions and blendshapes.
5. Export as VRM using a VRM addon.
6. Save to the same `models` path.

## Recommendation

Start with VRoid Studio for version 1.

After the character feels right, use Blender for a more exact construction-company uniform, headset, tablet, and accessories.
