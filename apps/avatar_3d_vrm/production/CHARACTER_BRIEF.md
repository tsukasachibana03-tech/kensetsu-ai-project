# 3D Model Character Brief

## Character

Name: Chibaten Field Secretary

Role:
- Desktop AI assistant for construction-office work
- Schedule, estimate, quantity takeoff, document, and conversation support

Visual:
- Young Japanese anime-style 3D woman
- Warm brown ponytail
- Headset microphone
- Teal and white construction-office jacket
- Dark inner shirt
- Light gray utility pants
- Teal and black work sneakers
- Tablet in left hand
- ID card and utility pockets

Personality:
- Friendly
- Calm
- Professional
- Slightly youthful
- Natural Japanese with soft Okinawan warmth

Required VRM Features:
- Humanoid rig
- Blink expression
- Smile expression
- Open-mouth talking expression
- Soft expression
- Surprised expression
- Basic idle pose

First priority:
- Clean full-body VRM model
- Stable face and mouth movement
- Lightweight enough for desktop use

Avoid:
- Existing anime/game character resemblance
- Logos
- Copyrighted marks
- Weapons
- Excessively revealing clothing
