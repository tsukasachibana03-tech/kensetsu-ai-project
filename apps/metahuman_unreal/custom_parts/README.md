# Chibaten Custom Outfit Proxy

This folder contains the first custom outfit/accessory proxy for the adopted Chibaten secretary direction.

Generated parts:

- Red cropped jacket
- Black high-neck fitted dress/skirt
- Black boots
- Cross necklace
- Earrings
- Dark purple hair silhouette for preview only

Output folder:

`outfit_proxy_v1`

Files created by Blender:

- `chibaten_custom_outfit_proxy_v1.blend`
- `chibaten_custom_outfit_proxy_v1.glb`
- `chibaten_custom_outfit_proxy_v1.fbx`
- `chibaten_custom_outfit_proxy_v1_preview.png`

Use:

1. Run `CREATE_CHIBATEN_OUTFIT_PROXY.bat` to regenerate.
2. Run `OPEN_CHIBATEN_OUTFIT_PROXY_IN_BLENDER.bat` to inspect in Blender.
3. Run `IMPORT_CHIBATEN_OUTFIT_PROXY_TO_UNREAL.bat` to import the FBX into Unreal.
4. Use the FBX/GLB as the starting point for fitting clothing to the MetaHuman body.

Unreal import destination:

`/Game/Chibaten/CustomOutfit/SM_Chibaten_CustomOutfitProxy_v1`
