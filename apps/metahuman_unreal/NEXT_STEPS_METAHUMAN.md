# MetaHuman Next Steps

このフォルダは、チバテンAI秘書プロジェクト用のMetaHuman作業場所です。

## 起動

1. `OPEN_METAHUMAN_PROJECT.bat` を開く
2. Unreal Engineが起動するまで待つ
3. Epicアカウントのログインが出たらログインする

## Epic Games Launcherで確認するもの

Unreal Engine 5.8 の `Options` から、MetaHuman Creator用データが入っているか確認します。

- MetaHuman Creator Core Data
- MetaHuman Creator / MetaHuman関連コンテンツ

入っていない場合はチェックを入れて `Apply` で追加します。

## Unreal Engine内の操作

1. `Edit > Plugins` を開く
2. `MetaHuman` で検索
3. `MetaHuman Creator` が有効か確認
4. 再起動を求められたら再起動
5. Content Browserで右クリック
6. `MetaHuman > MetaHuman Character` を作成
7. 名前を `ChibatenSecretary` にする
8. ダブルクリックして編集を開始

## 目標

VRoid版の秘書キャラを、MetaHuman品質のリアル寄り3Dアバターへ近づけます。
