# Mitsumori App

既存の見積りアプリを引き継いだフォルダです。

## 現在できること

- 見積書の作成
- 工種別の明細管理
- PDF / Excel / CSV の取り込み
- PDFプレビューと印刷
- Excel出力
- コンクリート、土工、型枠、左官、金属などのテンプレート利用

## 次に直すこと

- 日本語表示の文字化け修正
- PDF図面ビューアの追加
- 縮尺設定
- 距離計測
- 面積計測
- 拾い出し数量を見積明細へ反映

## 起動方法

`index.html` をブラウザで開くと動作します。

将来的には、PDF図面から数量を拾い出す積算AIの画面をこのアプリに追加します。
